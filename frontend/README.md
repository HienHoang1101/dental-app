# Frontend - Dental Clinic AI

Next.js 14+ application với TypeScript, TailwindCSS và shadcn/ui.

## 🚀 Getting Started

### Prerequisites

- Node.js 18+
- npm hoặc yarn

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Mở [http://localhost:3000](http://localhost:3000) để xem ứng dụng.

### Build

```bash
npm run build
npm start
```

## 📁 Cấu trúc thư mục

- `src/app/` - Next.js App Router pages
- `src/components/` - React components
- `src/lib/` - Utilities, hooks, API clients
- `src/types/` - TypeScript type definitions
- `src/store/` - State management
- `src/styles/` - Global styles

## 🎨 UI Components

Sử dụng shadcn/ui components. Để thêm component mới:

```bash
npx shadcn-ui@latest add button
```

## 📝 Environment Variables

Copy `.env.example` thành `.env.local` và cập nhật các giá trị.
